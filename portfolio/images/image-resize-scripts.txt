In folder originals
    magick convert *.jpg -resize 20% ../fulls/%01d.jpg
    magick convert *.jpg -resize 20% ../thumbnails/%01d.jpg

File size is reduced from ~10mb to ~500kb.

In corresponding folders (for example thumbnails)
    magick mogrify -strip -interlace Plane -quality 70% *.jpg

File size is reduced from ~500kb to ~70kb.
